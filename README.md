# Quizify

## General Information

Quizify enables users to login to our page, make any number of quizes on any topic of their choosing, and take quizes that others made

## Technologies Used

- Node.js
- Express.js
- RESTful API.
- Handlebars.js
- MySQL2 and the Sequelize ORM for the database.
- GET route
- POST route
